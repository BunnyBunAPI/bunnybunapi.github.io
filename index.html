BunnyBun

An isolated, free, and limitless registrar search engine built specifically for Porkbun extensions. BunnyBun pulls registry records cleanly using direct browser endpoints, ensuring your widget stays up 100% of the time without relying on an external backend.

Want to embed the separate BunnyBun widget code?

Copy the single line of code snippet underneath and paste it into any custom HTML block element editor to add this standalone search bar to another site.

<script src="https://YOUR_GITHUB_USERNAME.github.io/YOUR_NEW_REPO_NAME/widget.js" defer></script>
